When I figured out how to make websites look decent without using CSS, which was common practice back then as CSS wasn't really common knowledge, I decided that I must give this a try and dive into the wonderful world of retro web design.

I also tried to learn some stuff and get enlightened an inspired by old webpages such as Space Jam and some old snapshots of new websites using The Internet Archive which really built my taste in old web design, and I think that's fairly obvious now!

I don't know if this is just me, but something about these old websites or old software in general seems to have this cozy, warm and kind of nostalgic feeling even though I still didn't get to experience it.

It just feels much simpler than today's websites
